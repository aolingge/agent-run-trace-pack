# Security Policy

Report vulnerabilities privately to the maintainer instead of opening a public issue.

Do not include real credentials, cookies, browser profiles, private logs, private URLs, or customer code in reports. If a trace pack demonstrates a problem, recreate it with synthetic data before sharing.

Agent Run Trace Pack redacts common token-like values, but redaction is not a substitute for reviewing output before publication. Rotate any credential that may have appeared in terminal output before redaction.
